# TechPoint: Gestor de Rentabilidad

## Descripción del Proyecto

TechPoint es un sistema de gestión de rentabilidad diseñado para registrar, cotizar y analizar productos tecnológicos. Permite calcular márgenes de ganancia, gestionar precios y generar reportes de rentabilidad para la toma de decisiones comerciales.

## Stack Técnico

| Tecnología | Uso |
|---|---|
| **Java** | Lenguaje principal de desarrollo |
| **MySQL** | Base de datos relacional |
| **MVC** | Patrón de arquitectura de software |
| **PUD** | Metodología de desarrollo (Proceso Unificado de Desarrollo) |

## Metodología

El proyecto sigue la metodología **PUD (Proceso Unificado de Desarrollo)**, que organiza el trabajo en fases iterativas e incrementales:

1. **Inicio** — Definición del alcance y viabilidad
2. **Elaboración** — Análisis, arquitectura y planificación
3. **Construcción** — Desarrollo e implementación
4. **Transición** — Pruebas, despliegue y entrega
